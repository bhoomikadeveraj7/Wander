# 🌍 Wander – Know the Sky Before You Go

A modern and responsive travel website that helps users discover beautiful destinations while checking real-time weather conditions before planning their journey.

## 📖 About

**Wander** is a front-end travel web application designed to make trip planning easier and more enjoyable. The website combines destination exploration with weather information so users can make informed travel decisions.

The project features a clean, responsive interface with interactive animations, a destination carousel, and a weather search section, providing a seamless user experience across desktop, tablet, and mobile devices.

- 🏞️ Interactive destination carousel
